# 校招测评练习题库

这是一个纯静态手机端刷题 PWA，当前包含处理后的题库 JSON 分片和题图资源，不包含原始压缩包。

线上地址：https://yunzhixu620-stack.github.io/campus-question-bank-pwa/

## 当前数据

- 总题数：3360
- DOCX 批量题库：797
- PDF EPI 批量题库：2075
- 智鼎图片套题：488
- 数据分片：26

## 功能

- 手机端练习、每日计划、自由加练 30/40/50 题
- 错题本、收藏夹、最近练习
- 闪卡背题和自动轮播
- 模拟计时、答题卡、交卷、分数
- 搜题，支持题干、选项、解析、来源和 OCR 字段
- 草稿纸和简易计算器
- IndexedDB 本地记忆，支持 5 个本地用户档案
- Service Worker 离线缓存，打开过的数据可离线使用

## 发布说明

本仓库通过 GitHub Actions 部署到 GitHub Pages。发布内容只包含网页代码、处理后的题库分片和题图资源。

不要把原始题库压缩包、导入脚本、临时抽样文件或本地路径放进发布仓库。

## 已知限制

- 图片套题当前为整题截图资料题，截图里可能已有勾选或高亮答案，因此只用于搜题定位、收藏和背题浏览，不进入普通刷题。
- 已有 56 道图片题补入离线 OCR 文本用于搜索；其余图片题 OCR 和结构化解析待后续补充。
- 每日计划、随机练习、模块刷题和模拟考试都会避开 `image-only` / `reference-only` 图片题，只从有结构化选项和答案的题中抽题。
- 不做云同步，进度只保存在当前设备浏览器中。
